Easy Integration

Collect payments with Stripe

Add subscriptions to your app with just a few lines of code. Stripe handles billing, invoices, and compliance worldwide.

checkout.js
// Create a Stripe Checkout session for subscriptions
const session = await stripe.checkout.sessions.create({
  mode: 'subscription',
  line_items: [{
    price: 'price_voyager_monthly',
    quantity: 1,
  }],
  success_url: 'https://llama.inc/welcome',
  cancel_url: 'https://llama.inc/pricing',
});

// Redirect to Stripe's hosted checkout page
redirect(session.url);
